## 🖥️ كيف تستخدم المواد

كل جلسة عبارة عن ملف HTML واحد يحتوي على وضعين:

- **🎥 وضع العرض (Presentation Mode):** شرائح ملء الشاشة للتقديم الحضوري
- **📖 وضع القراءة (Reading Mode):** كل الشرائح كصفحة قابلة للتمرير + ملاحظات المدرب

### التحكم بالكيبورد (وضع العرض)

| المفتاح | الوظيفة |
|---------|---------|
| `→` أو `↑` | الشريحة السابقة (RTL) |
| `←` أو `↓` أو `Space` | الشريحة التالية |
| `F` | ملء الشاشة |
| `Home` / `End` | أول/آخر شريحة |
